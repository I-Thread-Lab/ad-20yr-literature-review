读完 140 篇之后的 5 点判断

  1. 分子端的胜利已经接近天花板。Aβ 抗体让数十年争议落地，但 27%–35% 的减速是当前生物上限；BACE、γ-secretase、N-端 anti-tau 全军覆没；GLP-1（EVOKE 2025）是下一个赌注。
  2. 诊断端的胜利刚刚到来。Plasma p-tau217（Palmqvist 2025, Nat Med）AUC ≥0.93，2024 NIA-AA 框架已经把血液 marker 接纳为 Core-1。AD 在生物学上已经"看得见"了。
  3. 但"行为/居家层"还是空白。Apple/Biogen/Stanford Nat Med 2025 用智能手表证明 n=23,000 消费级设备能筛 MCI，这是迄今唯一进入 Nature 系的可穿戴 AD 论文。多模态、生理耦合、闭环干预的可穿戴尚无 Nature 级论文。
  4. 多智能体临床 AI 才刚出现。DeepRare（Nature 2025）/ CARE-AD（npj DM 2025）都在静态 EHR/HPO 上推理，没有人把多智能体接到连续 sensor 流 + 生成式患者模型。
  5. "闭环可穿戴 RCT"是真正的空白。BPSD/sundowning 检测做到了（JMIR Aging 2025 提前 6 分钟），干预 RCT 做到了（音乐疗法 Cochrane 2025），两边没人合拢。
核心观察：过去 20 年所有顶刊的累积让我们能写出一份完整的 AD 患者"世界模型"——但没有人把这个世界模型从纸上的级联图变成一个可计算、可个体化、可被可穿戴驱动、可输出干预动作的工程系统。这是 2026 年最大的白空间。

二、AD 患者的世界模型（把 140 篇文献叠成一张图）

"世界模型"在这里的意思：把 20 年所有顶刊的因果证据合并成一份能 forward simulation 的患者模型——给它一个 70 岁、APOE4、HRV 异常、睡眠片段化的状态，它能 rollout 出未来 5 年的轨迹。2.1 是时间维度，2.2 是层次维度，2.3 是从两张图收敛出的硬事实。

2.2 因果链：五层耦合系统

这五层是怎么来的？ 不是先验拍脑袋——是把 140 篇文献按观测尺度自下而上聚类得到的：L1 分子层 ≈ 35 篇 (机制 / bench 顶刊) · L2 细胞层 ≈ 22 篇 (snRNA-seq / 小胶质 / 选择性脆弱) · L3 系统层 ≈ 30 篇 (临床纵向 + 可穿戴 + 数字健康，这是手套可读的层) · L4 行为层 ≈ 20 篇 (BPSD / 居家监测) · L5 社会层 ≈ 12 篇 (流行病学 / 护理 / Lancet Commission)；跨层综合 ≈ 20 篇 (世界模型 / 综述 / 多智能体)。 层间箭头方向来自纵向级联文献 (DIAN 2012 · AT(N) 2018 · Lancet Comm 2024) 与因果建模文献 (Green Nature 2024 细胞群体因果分离 Aβ vs tau · Seo Science 2023 ApoE×菌-tau 因果)。读图方式：每层 = 一个观测尺度；层内 = 机制 + 锚点；层间 = 上游对下游因果输出 (▼)；L5 → L1 有反馈 (公共卫生干预 → 血管/炎症/认知储备 → 分子负担)，所以级联是闭环。
L1 · 分子 MOLECULAR Aβ → tau → neurodegen 级联

AD 在分子层先表现为 Aβ42 错折叠 → 寡聚体 → 斑块沉积；随后触发 tau 磷酸化 → 神经原纤维缠结 (NFT)。睡眠驱动的 glymphatic 清除是关键调节器——睡得不好 = Aβ 清不掉。Lecanemab / Donanemab 在这一层赌赢了（清掉 76% 斑块、减速 27–35%），但 27–35% 是当前生物上限。

2.3 文献综合出来的 5 个"硬事实"

  1. AD 在临床确诊前 25 年就已经在分子上启动（Bateman DIAN 2012, Preische 2019, Buracchio 2010）。
  2. Aβ 是必要不充分。所有 anti-Aβ 单抗只能慢 27%–35%（Lecanemab CLARITY-AD; Donanemab TRAILBLAZER）。下游 tau / 神经炎症 / 突触丢失是另一条战线。
  3. 风险因素是可干预的。Lancet Commission 2024 把 14 个因素的 PAF 算到 45%。POINTER 2025 在美国队列验证了 FINGER。
  4. 诊断范式已经从临床切到生物。NIA-AA 2024 把血液 p-tau217 / GFAP 接纳为 Core-1。Plasma p-tau217 AUC ≥0.93。
  5. "行为/居家"这一层在分子和血液之间还是断裂的。它在数据上最丰富（手机/手表 24/7），但在 Nature 顶刊里还几乎是空的。

2过往文献的回答 · 不足与研究空白
  • 单模态 + 大规模派（Apple/Biogen, Yang voice, Apple iPhone-Stanford）: 证明单模态在大样本可达 MCI AUC ~0.8，但不能解释模态之间的耦合是否携带独立信息——本质是 univariate 假设。

  • 多模态机制派（Xie 2013, Spira 2013）: 给出多模态生理学因果耦合的机制基础（睡眠 → Aβ 清除是 sleep × CSF × autonomic 三者耦合），但没翻成 sensor-level coupling 特征工程。

3理论视角与框架

双理论框架:

  • Allostatic Load Theory（McEwen Ann NY Acad Sci 1998; Juster Lancet 2020 综述）: AD 是慢性应激-修复系统失调的累积结果。4 个 sensor 通道各自反映一个 allostatic mediator——HRV = autonomic balance; EDA = sympathetic arousal; IMU = sensorimotor regulation; voice = cognitive control。Allostatic load 的核心论点是: 失调先从 mediator 间耦合断裂开始, 再到单 mediator 显性失败。所以 prodromal AD 应首先在耦合特征而非单通道上显现——这是 H1 的理论根。
  • Embodied Cognition（Glenberg WIREs Cogn Sci 2010）: 认知衰退不是单纯"大脑问题", 是嵌入在 sensorimotor + autonomic system 的整体表现。多模态耦合直接读这个 embodied 状态, 单模态不能——这是 H3 的理论根。

理论框架图：

{IMU, PPG-HRV, EDA, voice} → 18 种耦合特征（cross-channel coherence / phase locking / lag correlation / mutual information）→ AD state（CU / SCD / MCI / mild AD）

4研究假设（逐条 + 推导）
H1主假设：4 通道耦合特征对 prodromal AD 的分类 AUC 显著高于最佳单通道特征（ΔAUC ≥ 0.05, DeLong p < 0.01）。
推导：直接来自 allostatic load——耦合断裂先于单 mediator 失败。
H2窗口假设：耦合 gain 在 SCD + early MCI 段最大（ΔAUC ≥ 0.10），在临床 AD 段减小（ΔAUC ≤ 0.05）。
推导：AD 神经退化的分布式起点假设——早期是 distributed 异常（多 mediator 微弱耦合断裂），后期收敛到 cortical motor + cognitive 显性失败（单通道已饱和，gain 自然小）。
H3机制假设：耦合特征与 plasma p-tau217 / NfL 的相关性（r ≥ 0.4）高于任何单通道（r ≤ 0.3）。
推导：embodied cognition——分子级病理通过 multi-system 而非单 system 表达。
H4临床转化假设：耦合特征对 18 个月内进展到 MCI 的预测优于单通道（Cox PH 耦合 HR ≥ 1.5, 单通道 HR ≤ 1.2）。
推导：H1 + H2 在纵向时间上的延伸。
H5规模换模态假设：n=137 + 合成数据 + WearGait-PD pretrain 的多模态耦合 AUC ≥ Apple/Biogen n=23K 单模态 AUC。
推导：信息论——每加一个独立模态约等于增加 log(n) 量级的样本信息（在通道独立性假设下）。

被试: Study 1 子集 n=50, 配对 plasma p-tau217 / NfL 抽血（Imperial NHS 合作 / BioFINDER 数据 sharing）。
变量: 同 Study 1 sensor 特征 + plasma marker。
统计: partial correlation (控制年龄 / 性别 / APOE4 / 教育)；hierarchical regression: 单通道 → 加耦合 → 看 R² 增量 ΔR² 显著性 (F-test)。
Study 3 — 纵向预测（验 H4）
被试: Study 1 中 baseline 为 CU+SCD 的子集 (估 n ≈ 60), 18 月随访诊断切换为 MCI。
变量: baseline 耦合 vs 单通道作为预测器, 终点 = 时间到 MCI。
统计: Cox proportional hazards, HR + 95% CI; Harrell C-index 比较; net reclassification improvement (NRI)。
Study 4 — 规模换模态 in silico（验 H5）
材料: bootstrap subsampling 你的多模态数据 (n=20 / 50 / 100 / 137 + synth) vs 用 Apple-style 单模态 (PPG+ACC only) 在公开数据集 (e.g. UK Biobank, n > 100K wearable) 重训。
统计: learning curve analysis, sample efficiency = AUC / log(n); 找两条曲线交叉点 → 等效样本量。
6预期实证发现 + 如何支撑理论
  • H1 预期 ΔAUC = 0.06–0.10（如单通道最佳 0.78 → 耦合 0.86）→ 支撑 allostatic load "coupling-first" 论点。
  • H2 预期: prodromal 段 ΔAUC ≈ 0.12, AD 段 ΔAUC ≈ 0.03 → 支撑 AD 分布式起点假设。
  • H3 预期: 耦合特征 vs p-tau217 r = 0.4–0.5, 单通道最佳 r = 0.2–0.3 → 支撑 embodied cognition——分子病理在 multi-system 上表达。
  • H4 预期: 耦合 HR = 1.7 (95% CI 1.3–2.2), 单通道 HR ≤ 1.2; C-index 提升 ≥ 0.05 → 临床有用证据。
  • H5 预期: n=137 多模态 AUC ≈ n=5,000–10,000 单模态 AUC（learning curve 交叉点）→ "质量打败规模"经济论证成立。

理论闭环: H1+H2 证明 allostatic load 适用于 wearable AD; H3 证明 embodied cognition 在 molecular layer 有具体表达; H4 把理论翻成临床预测器; H5 提供工程经济论证。四条假设全部支撑 = 论文论点闭环。

7贡献 · 局限 · 未来方向

理论贡献: 第一次把 allostatic load 框架从 cortisol / blood pressure 扩到 multi-sensor coupling, 给 AD prodromal monitoring 提供新理论基础。把 embodied cognition 从认知科学带到 wearable engineering。

方法贡献: 18 种生理耦合特征 + nested CV pipeline 开源；WearGait-PD → AD 的 transfer learning 模板；合成数据增广 pipeline (AD generator v2.1) 开源。

实践贡献: 证明"小样本多模态可以打败大样本单模态"——改变 wearable AD 资源分配范式；把数字健康从消费级带到医疗级证据门槛；为 LMIC 不需要科技巨头 cohort 也能做 AD 研究提供 evidence。

局限:

  • n=137 + 合成不够大, 外部 validity 有限（需后续 n > 1000 多中心复现）
  • 单中心, 地理 + 种族 + 文化 multiplicities 未控（Imperial + 中国 cohort 已为下一篇准备）
  • AD generator v2.1 walking bug 未完全修复（WearGait-PD pretrain 缓解但未消除）
  • 无 PET ground truth, 依赖临床诊断 + p-tau217（可接受替代但不是金标准）
  • BPSD endpoint 不在本文（留给 v3）
  • 因果方向: 耦合断裂 vs AD 是相关还是因果, 本研究是观察设计, 不能区分

未来方向（接 §2.5 roadmap）:

  1. v1.5 (Idea 5 + 4): 把多智能体 + 生成式世界模型接到耦合特征上, 做 forward simulation
  2. v2.0 (Idea 2): 用耦合特征做 inverse mapping 到 p-tau217 / Aβ-PET
  3. v3.0 (Idea 3): 把耦合检测接到闭环干预 RCT（直接因果验证）
  4. 亚型层面: 不同 AD 亚型（Q7）是否有不同的耦合签名（§2.4 gap ⑤）
  5. 多中心 + LMIC 复现（§2.4 gap ⑦）
  6. 跨病种: 同样耦合框架在 PD（用 WearGait-PD）、FTD、Lewy body 是否泛化

三、20 年里程碑时间线（详注版）

每一条注解告诉你"发生了什么、为什么重要、它让后面的什么变得可能"。生物学里程碑保持简洁，★ 标记的是与可穿戴/多智能体方向直接相关的关键节点，做详细注释。

2002 · Science Hardy & Selkoe: 修订版 Amyloid Cascade 假说

把 Aβ 失衡定为级联上游、tau 和神经元死亡为下游。这是 20 年 AD 药物研发的总剧本——所有 anti-Aβ 抗体（Aducanumab/Lecanemab/Donanemab）都源自这一假说。

2004 · Ann Neurol Klunk: PiB Aβ-PET 首次活体成像

在 AD 患者额叶看到 1.94× 的 PiB 摄取。在此之前 AD 病理只能尸检验证。这一篇是后来 AT(N) 框架、Lecanemab 入组标准、Donanemab 试验的影像基础。

2009 · J Biol Chem Frost & Diamond: tau prion-like 传播假说

细胞外 tau 聚集物能种入相邻细胞，引发胞内纤维化。把 tau 从"神经元末期产物"重新定义为"可传播种子蛋白"。2012 年 de Calignon 在 P301L 小鼠脑内证实活体传播，最终孕育了 BIIB080 这类 anti-tau ASO 治疗。

2012 · NEJM (DIAN) Bateman: AD 临床前期级联时间表 (25/15/10 年)

通过 PSEN1/PSEN2/APP 突变携带者，第一次量化了 "症状前 25 年 CSF Aβ 下降、15 年海马萎缩、10 年认知改变"。"临床前 AD"概念从此被科学化。这是后来 Lecanemab/Donanemab 把治疗推前到 prodromal 阶段的科学基础。

2012 · Sci Transl Med + 2013 · Science Iliff / Xie / Nedergaard: glymphatic 系统 + 睡眠驱动 Aβ 清除

大脑通过血管旁 AQP4 水通道实现 CSF-ISF 交换清除 Aβ；睡眠时这一过程被放大 60%。把睡眠从 AD 的"症状"重新定义为"上游可干预因子"。后续所有可穿戴睡眠监测 AD 工作的机制基础。

2013 · NEJM Guerreiro & Jonsson: TREM2 R47H 罕见变异 → AD 风险 3×

微胶质先天免疫成为 AD 核心通路。从 Aβ 中心主义打开新战线。2017 年 DAM 微胶质分型和 2022 年 Bellenguez GWAS 都建立在这条线上。

2015 · Lancet Ngandu (FINGER): 多领域生活方式 RCT 首次阳性

饮食 + 运动 + 认知训练 + 血管监测组 NTB +25%。第一个证明 AD 可被生活方式预防的 RCT。催生 World-Wide FINGERS 全球复现网络 + 2025 年美国 POINTER 验证。

2016 · Nature Sevigny (Aducanumab PRIME): 抗体清斑首次人体证据

PET 确证剂量依赖、时间依赖的 Aβ 斑块清除。20 年 anti-Aβ 押注第一次有人体阳性——尽管后续 EMERGE/ENGAGE 结果矛盾，最终撤回，但它打通了 Lecanemab/Donanemab 的路。

2016 · Science Hong (Stevens lab): 补体 + 微胶质介导早期突触丢失

C1q/C3/CR3 抑制可救援 Aβ 寡聚体诱导的突触修剪。突触丢失第一次有明确细胞机制。后续 anti-C1q 药物（如 Annexon ANX005）来源于这条线。

2017 · Cell + Lancet Keren-Shaul: DAM 微胶质分型 + Livingston: Lancet Commission 9 因素

细胞层和公共卫生层同年成立两个框架——DAM (Disease-Associated Microglia) 是 TREM2 依赖的两步活化程序；Lancet Commission 第一次把 9 个可干预风险因素的 PAF 量化到 35%。分子和人群两端的因果建模同步起步。

2018 · Alzheimer's & Dementia Jack et al.: NIA-AA AT(N) 框架

AD 第一次被"生物学定义"——Amyloid + Tau + Neurodegeneration。临床表征不再是诊断的必要条件。后来 2024 年的更新把血液 marker 接纳为 Core-1。

2019 · Nature + Nat Neurosci Mathys / Grubman: AD 单细胞 (单核 RNA) 图谱

80,660 核（Mathys, Nature）+ 内嗅皮层细胞特异性 APOE 调控（Grubman, Nat Neurosci）。病理走进单细胞分辨率，使后续 2021 RORB+ 神经元（Leng）、2023 GFAP+ 星形胶质 gating（Smith）成为可能。

2019 · Nat Med Arboleda-Velasquez (Lopera lab): APOE3-Christchurch 抗 PSEN1 AD 30 年

哥伦比亚 Paisa 家族携带 PSEN1 E280A 突变的女性同时纯合 APOE3-R136S，比家族其他人晚 30 年发病。人体首证：APOE 通路可被改造来推迟 AD。2023 年同组又发现 Reelin-COLBOS 第二个保护变异，启动 APOE/Reelin 通路药物研发。

2019 · NEJM × 多篇 Verubecestat APECS + Lanabecestat: BACE / γ-secretase 类药全军覆没

BACE1 抑制剂导致认知恶化、被迫停止；γ-secretase 也类似（Semagacestat 2013）。小分子上游策略基本死亡，剩下抗体 + ASO 两条线。

2020 · Nat Med + Lancet Janelidze: Plasma p-tau181 + Livingston: Lancet Commission 12 因素

血液诊断（p-tau181 AUC 0.89–0.94 区分 AD vs 非 AD）+ 预防框架（12 个因素 PAF 40%）同步升级。诊断和预防两端同时跨过新门槛。

2023 · NEJM van Dyck (CLARITY-AD): Lecanemab 27% 减速 CDR-SB

1795 例早期 AD 双盲 RCT，18 月减缓 27% 临床衰退。20 年 anti-Aβ 押注终于无歧义赢一次。FDA 全批准 2023-07，但 ARIA-E/H 15–35% 副作用 + 每两周一次输注让落地受限。

2023 · JAMA Sims (TRAILBLAZER-ALZ 2): Donanemab 35% 减速 iADRS

1736 例，低/中 tau 亚组减缓 35%，76% 患者达到斑块清除。第二个 anti-Aβ 胜利，FDA 全批准 2024-07。但配套要求繁重（每 6 月 PET 复查 + 每 3 月 MRI 看 ARIA），临床落地需要可穿戴/血液 marker 来辅助监测。

2023 · NEJM (A4) Sperling: Solanezumab 在临床前期失败

1169 例 amyloid+ 认知正常老人，240 周 PACC 无减缓。"越早越好"假说被部分证伪——只针对 soluble Aβ 不足，需要清斑级别的清除（Lecanemab/Donanemab）。

2024 · Nat Med Banerjee & Collinge: Iatrogenic AD 通过尸源 GH 传播

5 例 1958–1985 年用过尸源生长激素的患者出现早发 AD。AD 病理首次在人体内显示 prion 样传播能力。重新定义生物安全（外科器械、血液制品）框架。

2024 · Lancet Livingston (Commission): 14 因素覆盖 45% 痴呆风险

新增 LDL 胆固醇 + 视力损失。痴呆可预防的上限从 35% (2017) → 40% (2020) → 45% (2024) 不断上升，公共卫生干预的科学依据进一步加强。

2024 · Nat Med Barthélemy / Bateman: Plasma p-tau217 ≥ CSF

AUC ~0.94 匹配 FDA 批准的 CSF 检测。腰穿可以退场了。2024 年 NIA-AA 框架正式把血液 marker 列为 Core-1。

2024 · Nat Aging Guo (UK Biobank): 血浆蛋白组预测 14 年痴呆

n=52,645 队列，GFAP/NEFL/GDF15/LTBP2 预测 14 年痴呆 AUC 0.891。人群级"提前 10+ 年"的非临床预测做到了。但仍需采血——可穿戴层面还没人做到这个 lead time。

2025 · Nature ★ Zhao (DeepRare): 多智能体临床 AI 第一次进 Nature

详注：DeepRare 是一个面向罕见病诊断的多智能体系统，由一个 central host 协调 40+ 工具（HPO 表型库、文本检索、基因库、文献检索）。在多个外部测试集上首次尝试准确率 64.4%，超过资深医生的 54.6%；临床医生对 95.4% 的推理链表示认同。它部署在 600+ 机构。意义：(1) 多智能体临床 AI 首次以"可追溯推理"为核心论点进 Nature 主刊；(2) 它工作在静态 HPO + 文本输入上——患者表型描述是固定的快照。(3) 未做到的事：没有连续多模态 sensor 流输入、没有生成式患者模型做反事实推演、没有干预动作输出。这定义了"可追溯多智能体"赛道的当前天花板，也告诉我们空间在哪里。

2025 · npj Digital Medicine ★ Li (CARE-AD): 多智能体在 EHR 注释上预测 AD 提前 10 年

详注：CARE-AD 部署专科 LLM agent (神经/精神/老年/全科) 团队，在纵向 EHR 注释（不是 sensor 流）上协同推理，预测 10 年后 AD 发病。准确率 0.53 vs 单 agent baseline 0.26–0.45。意义：多智能体范式正式从罕见病迁到 AD。未做到的事：(1) 输入仍是医生写的临床笔记，被动反映"医生看见的事"——而 AD 患者一年只来 2–3 次门诊，绝大多数轨迹缺失；(2) 没有 sensor 数据；(3) 没有生成式患者模型做 forward simulation；(4) 没有干预输出。它定义了 multi-agent for AD 的当前位置，但留下的空白对可穿戴 + 世界模型的方向是完全空着的。

2025 · Nature Medicine ★ Chen / Dagum (Apple/Biogen/Stanford "Intuition"): 智能手表 + iPhone 远程 MCI 检测 n=23,000

详注（迄今唯一进入 Nature 系的可穿戴 AD 论文）：18 个月队列，使用 Apple Watch 和 iPhone 被动 (心率/活动/睡眠) + 主动 (任务) 信号训练 MCI 分类器。意义：消费级可穿戴第一次在 AD 上达到 Nature Medicine 的规模门槛——n=23K 远超传统医院级研究。未做到的事：(1) 只用消费级 sensor (PPG+IMU+加速)，没有 EDA、没有麦克风、没有跨模态生理耦合；(2) 单一终点 = MCI 分类，不做 BPSD 预警、不做干预触发；(3) 数据是被动观察，没有生成式模型反推隐藏状态；(4) 没有闭环。这定义了"可穿戴 AD"赛道的当前天花板：单模态行为信号 → 单点分类。多模态 + 生理耦合 + 闭环干预这三条线没人做过 Nature 级工作。

2025 · JAMA Baker (US POINTER): FINGER 在美国 n=2,111 复现

结构化生活方式组优于自助组。第一个美国大规模 FINGER 复现，把多领域干预推到国家政策级。

2025 · Nat Med Palmqvist (BioFINDER): Plasma p-tau217 全自动平台

Lumipulse 全自动免疫分析，AUC 0.93–0.96 准确率 85–91%，匹配 CSF。第一个 FDA 可清通的 AD 血液检测工作流。

2025 · Cell Xiong (Tsai lab): 多区域 AD 表观图谱

6 个脑区染色质可及性图谱揭示细胞类型特异性调控重塑跟随认知韧性。分子分辨率再升一级——从单细胞 → 单细胞 + 表观。

2026 · Nat Med Mattsson-Carlgren (BioFINDER): ProtAIDe-Dx 一管血鉴别 6 种痴呆

n=17,187 队列，7,595 蛋白联合学习同时分类 AD/PD/FTD/ALS/卒中/对照，AD 准确率 81%，全部 AUC>0.78。分子诊断从 "AD vs 非 AD" 推到 "AD vs 其他痴呆"鉴别诊断级。

2026 · Nat Med Janelidze: Plasma p-tau217 时钟预测发病年份

血浆 %p-tau217 轨迹预测 AD 症状起病年龄，中位误差 3.0–3.7 年。从 AD 二分类预测推到个体化预后时间预测。

2026 · Sci Data (Nature) ★ Watkinson et al. (MJFF / SAGE): WearGait-PD 开源多通道步态数据集 n=185

详注（与你 L3 方向直接相关）：100 PD + 85 年龄对照，每人 13 个 IMU 传感器 + 双足 16-传感器压感鞋垫 + 步行通道 2D 位置/压力 + 视频帧级动作标注，全部对齐 MDS-UPDRS / DBS 状态 / 用药信息。意义：这是当前唯一开源、Nature-tier、带细粒度临床标签的多通道步态数据集。虽然是 PD 不是 AD，但 (1) IMU 步态在 Buracchio Arch Neurol 2010 已证明 AD 早 12y 异常；(2) AD generator v2.1 当前的 b-class walking bug（ρ(jerk,HR)=0.13 vs 真实 0.51）正好需要 supervised 步态数据做 pretrain；(3) PD ↔ AD 鉴别诊断本身也是 ProtAIDe-Dx 在做的事。它给 L3 → L1 inverse mapping 提供了首个可直接使用的预训练底盘——PD 上 supervise，迁到 AD。

2026 (今) · 你的位置 手套 (EDA+PPG+IMU+mic) + 7-Agent 推理栈 + AD generator v2.1 + ProtAIDe-Dx

把上面这条 20 年时间线放到一张图上看，目前的占位是这样的：消费级单模态规模属于 Apple/Biogen (n≈23k 智能手表筛 MCI, Nat Med 2025)；静态多智能体诊断属于 DeepRare (Nature 2025, HPO+文本) 和 CARE-AD (npj DM 2025, EHR 笔记 4-agent 协同, AUC 0.53)；分子端属于 Lecanemab / Donanemab (清 76% 斑块, 减速 27–35%)。这三块从来没合到一个 stack 里。

我们在做的是把它们合起来——具体到当前的实施状态：
(1) 多模态 vs 单模态。手套同时拿 EDA / PPG / IMU / 麦克风 四路信号，对应 L3 系统层的四条通路 (自主、运动、心血管、语言) 一次性截下来，而不是只看一只手表的活动量。
(2) 连续 sensor 流 vs 静态 EHR。把 DeepRare / CARE-AD 的"多智能体推理"接到 24×7 流式输入上——目前栈里 7 个 specialist agent：神经 / 心血管 / 运动 / 睡眠 / 语言 / 精神 / 护理，每个 agent 监听对应通道，central coordinator 做跨 agent 推理。
(3) 生成式患者模型。AD generator v2.1 已经能用 5 个 OpenNeuro EEG 数据集 (ds002778 / 004504 / 007427 等, n=163) + WESAD / DaLiA / mHealth / PAMAP2 PPG+IMU (n=49, 42 hr) 做 forward simulation，给反事实"如果不干预会怎样"的 rollout 引擎。当前在修 B 类 generator 的 walking bug：ρ(jerk, HR) = 0.13 vs 真实 0.51；HR 66 vs 真实 94 BPM。
(4) 实际数据底盘。量表库 137 例 (经临床合作) + 传感器 baseline 4 例 + 公开数据集 10 个；已建立 17 个 AD signature + Transformer 架构 + FUPCC 评估框架。
(5) ProtAIDe-Dx。把 (1)–(4) 串到一个 sense → reason → simulate → intervene 的闭环里，目标是占住 Nature 系还空着的那块：L3 多模态 + L4 BPSD 预测 + 闭环非药物干预 这格。

说白了不是"再来一个 demo"。不在分子端（药企赌过了，27–35% 天花板）；不在静态多智能体（DeepRare 已占）；不在消费级单模态（Apple 已占）。位置是多模态 × 流式 sensor × 生成式世界模型 × 闭环干预这四者的交集——上面四块没人占。

四、AD 在医学上还没搞清楚什么（客观盘点）

这一节回答"AD 现在是不是已经搞清楚了"——文献证据显示，它远没有。下面列出当前主流共识承认的 8 个开放问题。

Q1 · Aβ 是 AD 的根因还是下游放大器？
主流仍接受 Amyloid Cascade 假说（Hardy & Selkoe 2002, 2016），但 Lecanemab/Donanemab 清掉 76% 斑块也只能减速 27–35%（NEJM 2023, JAMA 2023），说明 Aβ 是必要不充分。许多 Aβ 高负担的认知正常老人（A+T-）多年不发病，许多病理性 AD 的 Aβ 并不极端。当前正在重新评估：Aβ 是病因还是放大器？这关系到下一代靶点应该指向哪里。
Q2 · 为什么内嗅皮层先死，新皮层后死？
Leng 2021（Nat Neurosci）发现内嗅皮层 RORB+ 兴奋性神经元最先累积 tangle，但为什么是这一类细胞？是脂代谢易损（Tsai 2024 Nature）？是 Reelin 信号失调（Mathys 2024 Nature）？是网络拓扑使然？目前都是部分解释，无统一机制。直接影响"早期干预靶点"的选择。
Q3 · 微胶质是 AD 的好人还是坏人？
TREM2 R47H 提高 AD 风险 3×（Guerreiro/Jonsson 2013），DAM 微胶质（Keren-Shaul 2017）参与清除斑块——这两个发现方向相反。Wang/Colonna 2015 显示 Trem2-/- 微胶质无法聚集斑块并凋亡（暗示微胶质是好人）；Hong/Stevens 2016 显示补体 + 微胶质修剪突触（暗示坏人）。目前模型：早期保护、晚期有害——但转换点在哪、能否调控仍未知。
Q4 · 为什么女性 AD 更多更重？
全球女男比 ~1.7（Lancet PH 2022）。Yan 2022（Cell）发现 X-linked USP11 逃逸 X 失活、去泛素化 tau，提供机制基础。但是否还有雌激素轴、APOE-性别交互、寿命差异叠加？未明。直接影响"是否应该按性别建模 + 是否需要性别特异性干预"。
Q5 · 多少 AD 是可预防的，剩下不可预防的来自哪里？
Lancet Commission 2024 说 14 个因素覆盖 45% PAF。剩下 55% 来自什么？遗传（APOE/罕见变异）？随机蛋白错折叠累积？环境暴露未识别？这 55% 的成分会决定我们的研究是该往"预防"还是"早治"倾斜。
Q6 · 临床前 AD 的 25 年发生了什么？
Bateman DIAN 2012 给出"症状前 25 年 Aβ 下降、15 年萎缩、10 年认知变化"的级联，但这些步骤之间的因果关系、个体变异、亚型分化都是黑箱。同样 Aβ+ 的人为什么有的 5 年后发病、有的 30 年不发病？认知储备 + 微胶质响应 + 心血管健康都参与了，但权重未知。
Q7 · AD 是一个病还是多个亚型？
越来越多证据指向至少 3–5 个生物学亚型：典型海马型 vs 后部皮层萎缩型 vs 内嗅皮层主导型 vs Limbic-predominant AD 等。Gockley 2024（Nat Genet）空间转录证据进一步显示散发 vs APOE4 vs PSEN1 分子签名不同。如果 AD 是多亚型，治疗、监测、试验入组都需要重新分层。
Q8 · 行为/睡眠/运动指标究竟反映 AD 的哪一层？
Xie 2013（Science）证实睡眠驱动 Aβ 清除——所以睡眠紊乱是上游因子还是下游表征？步速下降（Buracchio 2010）比 MMSE 早 12 年——但反映海马、还是基底节、还是脑白质退化？可穿戴信号到底映射到哪一层病理至今未明，这正是 inverse mapping 的开放空间。
说人话 · 2026 年的 AD 到底是个什么状态？

说白了，这病现在的处境是这样：医学已经知道它怎么一步步坏下来——分子层 25 年前就开始 Aβ 堆积，然后 tau 缠结、神经元一批批倒下，最后大脑萎缩、人忘了自己是谁。这套级联在顶刊里画过上千遍。诊断这一端也终于跟上：一管血查 p-tau217 准确率超过 0.93，比 PET 便宜十几倍，2024 NIA-AA 已经把它写进核心生物标志。

治疗这一端 2023 年开始有了两个真药 (Lecanemab、Donanemab)——能清掉 76% 斑块，听起来很爽，但临床终点只能减缓 27–35% 的衰退速度，所以本质是减速器不是治愈。而且药很贵 (年费 2.6 万美元)、副作用 ARIA (脑水肿/微出血) 不算轻、必须在症状极早期 (MCI 到 mild AD) 才能用，APOE4 双拷贝的人风险还更大。再加上输液中心 + PET 筛查这些基础设施门槛，绝大部分国家用不起、用不上。

最现实的一层问题是：诊断后的日常没人接得住。全球 85% 患者拿到诊断之后没有持续随访；主要照护者一半进入临床抑郁；BPSD (激越 / 游荡 / 日落综合征 / 幻觉) 是入机构的实际触发点，没有任何 disease-modifying 药能阻止它们。这一段——从拿到诊断到末期——平均 10 年，主要发生在家里，不在医院、不在试验。

所以 2026 年的 AD 不是"完全没搞清楚"，也不是"已经搞清楚"。生物学上有清晰的级联框架 + 单细胞图谱 + 血液诊断，临床上有两个能减速 30% 的药，但 8 个核心机制问题没共识、治疗窗口很窄、居家这一层在 Nature 顶刊里基本是空的。"AD 是什么"的世界模型是一张半透明的网——粗框架清楚，节点和参数还在被填，行为/居家/闭环这一格几乎没人画。

五、治愈与监测：医学现状 + 不同类型工作能填的位置

5.3 一句话总结

药物端能减速但不能治愈，且只在 MCI–轻度阶段有效。诊断端已经有血液 marker 接近 CSF。预防端已经知道 14 个因素能挡掉 45% 风险。监测端是最年轻、证据最稀的一端——它能不能从"商业健身追踪"上升到"医疗级证据"，取决于多模态、生理耦合、闭环干预这三条线能不能在 Nature 级研究上落地。

5.4 监测到底有什么好处？（八条客观收益）

"为什么不只是给老人多戴一个手环"——这里把监测能带来的实际收益按能挽回什么 × 能值多少钱 / 多少时间列出来，每条都接上现有文献的证据强度。

A · 临床盲区填补

1. 把 25 年的盲区填上

DIAN 2012 说 AD 在临床确诊前 25 年就启动，但门诊一年只看 2–3 次——99% 时间临床是盲的。可穿戴是唯一能 24×7 把这 25 年填满的工具。p-tau217 测一次知道"现在是不是 +"，但说不出"过去 5 年怎么变的"——这正是监测的位置。

B · 治疗窗口

2. 让更多人挤进 Lecanemab/Donanemab 窗口

这两个药只在 MCI–mild AD 有效。但目前症状被注意到 → 就诊 → 确诊平均延迟 2–5 年，很多人到诊时已经过了窗口。监测能在 SCD/前驱期就提示"你正在向 MCI 滑"——把可治人群放大几倍。Apple/Biogen Nat Med 2025 已经在 n=23k 上验证手表能筛 MCI 候选。

C · BPSD 预警

3. BPSD 提前 6 分钟 → 提前几个月

JMIR Aging 2025 用 EDA+HR+ACC 提前 6 min 预警 agitation；Iaboni 2022 AUC 0.87。即使只把这件事工程化到家里——延迟入机构 6–12 个月——对单个家庭 = 几万美元，对医疗系统 = 几十亿美元级。这是离 RCT 最近、ROI 最高的一格。

D · ARIA / 药物监测

4. Lecanemab/Donanemab 的安全监测

这两个药每 3 个月要 MRI 看 ARIA-E/H、每 6 个月要 PET 复查。临床实际操作很重。可穿戴 + 血液 marker 组合能把窗口压缩到家里，让"药 + 监测"成一个产品，而不是两套独立的服务。这是药企已经在找的合作位。

E · 个体化

5. 同样 Aβ+ 为什么有人 5 年发病 有人 30 年不

这是 8 个开放问题之一 (Q6)。原因可能在 L3 (认知储备 / 微胶质响应 / 心血管健康) 的个体级协变量——但这些都是系统层信号，正是可穿戴在测的层。监测产生的纵向数据 = trial enrollment + dose-response 的分层依据。

F · Inverse mapping

6. 反推分子层 → 降低筛查门槛

开放问题 Q8 直接问："步速 ↓ 反映海马、基底节、还是脑白质？"。如果手套的 jerk/HRV 能反推 p-tau217 / Aβ-PET 分布，意味着不抽血不影像也能做大规模筛查——对发展中国家 (诊断率 <10%) 是关键的门槛降低。Apple 2025 已经走到第一步。

G · 非药物 RCT

7. 给 Lancet "45% PAF" 提供工程化路径

药物只能 30%；Lancet Commission 2024 算出 14 个可干预因素能挡 45%——但目前都是观察证据。监测 + 闭环干预 (光、声、运动、社交) 是把这 45% 从流行病学搬到 RCT 的唯一工程化路径。Cochrane 2025 已经证明音乐对激越有效，但还是 caregiver-initiated；自动触发的 RCT 还是空的。

H · 多智能体燃料

8. 给生成式世界模型 + multi-agent 喂数据

DeepRare / CARE-AD 都因为缺连续 sensor 数据所以只能在快照上推理。监测产生的纵向多模态流 = 生成式患者模型 (counterfactual rollout) 的训练 fuel，也是多 agent 协同推理的输入。这是把"静态多智能体"升到"动态闭环"的唯一路径。

一句话： 监测不是"再加一个手环"——它是填补 25 年临床盲区、把治疗窗口和 BPSD 干预窗口拉开、给非药物 RCT 提供工程路径、给多智能体喂连续数据这四件事的唯一可行底盘。从商业健身追踪到医疗级证据的跨越，需要的就是多模态 × 生理耦合 × 闭环干预这三条线在 Nature 级研究上同时落地——这是 2026 年最大的可占空间。

十、白空间地图（2026 年还空着的位置）

把上面 4 张表叠起来，能看出 4 个明显的空位。每个空位都标了"目前谁最接近、距离多远"——客观盘点，不假定具体方向。

WS-1 · 多模态 AD 世界模型

最接近：DADD 2025（仅 EEG）/ Unlearn CRBM 2025（仅 baseline，不可 rollout 干预）/ MeWM 2025（肿瘤，不是 AD）

空缺：没有任何已发表工作是多模态 + cross-modal coupled + 可个体化条件 + 可反事实推演的生成式 AD 患者动力模型。LeCun JEPA / Sora 范式在 AD 上完全空白。

WS-2 · 多智能体 × 连续 sensor 流

最接近：DeepRare Nature 2025（HPO + 静态文本）/ CARE-AD npj DM 2025（EHR 注释）/ ADAgent 2026（影像 + LLM 协调）

空缺：所有现有 multi-agent AD 系统都在静态艺术品（笔记、影像、基因）上推理。没人把多智能体接到 7×24 连续多模态 sensor 流，且没人让 agent 调用世界模型做 counterfactual rollout。

WS-3 · 闭环可穿戴 RCT

最接近：JMIR Aging 2025（agitation 提前 6 min 检测）/ Cochrane 2025（音乐疗法 RCT，但 caregiver 触发）/ LightSPAN 2026（光照协议，未闭环）

空缺：检测端和干预端两条线 20 年来没有合拢。没有一个 RCT 是"sensor 自动检测 BPSD 触发 → autonomous 干预 → 量化 outcome"的全栈闭环。

WS-4 · 行为/居家层 → 分子端的可解释桥梁

最接近：Apple Intuition Nat Med 2025（n=23K 但只到 MCI 分类）/ UK Biobank Nat Aging 2024（蛋白组 10 年提前但需采血）

空缺：没人证明纯被动可穿戴信号能在不抽血、不影像的条件下，准确预测 p-tau217 / Aβ-PET 这些金标准。手套 → 分子层的 inverse mapping 是开放问题。

观察：4 个白空间互相耦合——WS-1（世界模型）给 WS-2（Agent）提供 rollout 引擎，WS-2 给 WS-3（闭环）提供决策器，WS-3 的真实回流又校准 WS-1。它们既可以分开做，也可以叠起来做。

十一、三个候选研究方向（客观对比）

基于上面 4 个白空间，可以收敛出 3 个互不重叠的候选方向。这里只列出每个方向的目标、可行性、潜在风险，不假定哪个最适合——这取决于具体团队配置、临床合作可用性、时间线偏好。

11.1 哪些位置已经被占满（不论选哪条都要避开）

  • 单模态 ML 分类器（步态/语音/睡眠单跑）：Apple Intuition / npj Aging sleep / Yang Commun Med 占满。
  • 新血液 biomarker 单跑：Hansson / Bateman 的 BioFINDER / DIAN 占满 p-tau217 / MTBR-tau243 / NfL。
  • 另一个静态多智能体诊断系统：DeepRare + CARE-AD + ADAgent 三连击。
  • 新单细胞图谱：Tsai / De Jager 三个 Nature/Cell。
  • 新 Aβ/tau 抗体：药厂主导。

11.2 三个候选方向

方向 A · The Glove-Driven AD World Model + Closed-Loop Agent

论题：构建首个由可穿戴信号驱动、可个体化、可反事实推演的 AD 世界模型，并把单只手套变成一个完成 sense → reason → intervene → re-measure 闭环的临床智能体。

占的白空间：WS-1 + WS-2 + WS-3 三块同时。

论文路径（3 篇打包）

  • P1 (Nat Med / Nat Commun)：Conditional Multimodal Diffusion-Transformer AD generator + blind clinical Turing test
  • P2 (Nature / Nat Med)：World-model-coupled multi-agent system on continuous sensor stream，MCI lead time vs Apple Intuition baseline
  • P3 (Lancet Dig Health / Nat Med)：n≈30 sundowning RCT，BPSD 频次 ↓40%

风险：(1) Paired 采集 (5–10 例) 是单点故障；(2) 需要药物/p-tau217 cross-validation；(3) 闭环 IRB 复杂；(4) 三篇耦合，任一失败影响整体 narrative。

时间线：12–24 月 + 临床合作 + 工程资源。

方向 B · The "Glucose Monitor for AD" — 可穿戴 → 分子 inverse model

论题：建立可穿戴信号到 p-tau217 / Aβ-PET / 海马萎缩的 inverse mapping，让手套变成 AD 的"血糖仪"——不抽血、不影像、连续看见。

占的白空间：WS-4 单块。

论文路径（1 篇主线 + 后续）

  • Main (Nat Med)：n≈50–100 paired wearable + p-tau217 + tau-PET 队列；模型预测分子标志物 AUC≥0.85
  • 后续可扩展到 trial enrichment 商业化路径

风险：(1) 需要相对大的 paired 队列 (50+) 才有统计力；(2) ceiling 高但依赖 paired 采集规模；(3) 不依赖闭环创新，narrative 单线。

时间线：18–30 月 + 多中心合作。

方向 C · HCI 端可追溯多智能体 + 临床落地 (深度而非广度)

论题：聚焦"可追溯多智能体 + 家属-医生沟通报告"的 HCI 部分，做 5–10 例 deployment study，证明系统在真实临床工作流中的可用性。

占的白空间：WS-2 的一个 subset（不接 world model，但深度 HCI 评估）。

论文路径：CHI / UIST / CSCW main conference paper + 可能 Lancet Dig Health。

风险：(1) 进不了 Nature 主刊；(2) HCI 主流 venue 接受度高但 scientific impact 单一；(3) 与已有 CARE-AD/DeepRare 同范式，差异化要靠 HCI 方法学。

时间线：6–18 月。
